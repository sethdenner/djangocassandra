# Knotis Mobile
## Requirements
The following packages are required to build Knotis Mobile

### nodejs

```sh
wget http://nodejs.org/dist/v0.10.22/node-v0.10.22.tar.gz
tar xvf node-v0.10.22.tar.gz
cd node-v0.10.22
./configure
make
sudo make install
```

### cordova

```sh
sudo npm install -g cordova
```
